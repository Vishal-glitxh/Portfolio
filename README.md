## ✨ Features

*   **Modern UI**: High-end cyberpunk/SaaS telemetry aesthetics featuring glass panels and glowing accents.
*   **Responsive Design**: Mobile-first fluid scaling across desktop, laptop, tablet, and mobile displays.
*   **Framer Motion Animations**: Micro-interactions, spring-based card tilts, and smooth layout transitions.
*   **React Three Fiber Integration**: Pre-configured support for 3D coordinate-tracking and rigging engines.
*   **Interactive Components**: Local AI chatbot assistant, Recruiter summary dashboard console, and sound mixers.
*   **Glassmorphism**: Glass panels styled with fine gradient borders, low-opacity backdrops, and backdrop filters.
*   **Education Timeline**: Redesigned animated vertical timeline with glowing pulsing nodes and color accent codes.
*   **Projects Showcase**: Custom interactive laptop mockups showing live project roadmaps and tags.
*   **Skills Showcase**: Technology catalog filter grid.
*   **Contact Section**: Direct mail forms with social telemetry links.
*   **Performance Optimized**: Bundles are code-split, and assets are compressed for fast load speeds.
*   **Accessibility Focused**: Built with keyboard navigation tab locks, focus states, and ARIA labels.
*   **SEO Friendly**: Structured meta descriptions, keywords, heading hierarchies, and OpenGraph variables.

## 🚀 Getting Started

### 1. Installation
Clone the repository and install all dependencies:
```bash
npm install
```

### 2. Development
Start the local development server:
```bash
npm run dev
```
Open `http://localhost:3000` inside your browser to see the live rendering.

### 3. Production Build
Compile and start the optimized production bundle:
```bash
npm run build
npm start
```

## ⚡ Performance Optimizations

*   **Optimized Images**: Formatted and compressed public assets to prevent layout shifts.
*   **Dynamic Imports**: Browser-only modules are loaded using Next's `dynamic` compiler with `{ ssr: false }` to guarantee fast initial paints.
*   **Lazy Loading**: Subsections and complex meshes are deferred until they enter the viewport bounds.
*   **Strict Typing**: Compiled with strict TypeScript checking to catch bugs at compile time.
*   **SEO Optimization**: Fully loaded keywords and description tags inside root layouts.
*   **Accessibility**: Semantic HTML structures with screen-reader compatibility tags.
